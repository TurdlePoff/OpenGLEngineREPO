Vivian Ngo - Summative 3 Graphics ReadMe

Implemented features:
	- Non GPGPU rain particles system with billboard quads
	- Animated character moves on terrain and has idle, run and jump animations
	- Toon shader applied to animated character and terrain

Controls:

**Player Controls**
	W - forwards
	S - backwards
	A - left
	D - right
	SPACE - Jump

**Camera Controls**
	Please hold down the middle mouse 
			SCROLL button
	to rotate the camera view + move with the following controls:
	U - forwards
	J - backwards
	H - left
	K - right

	
**Misc Controls**
	M - Toggle terrain mesh
